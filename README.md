# Pdf-Extension
An extension to download Pdf from any page you're visiting

## Installation

### For Chrome:
1. Download the [Extension ZIP](https://github.com/3issam-hub/Pdf-Extension.git)
2. Extract the ZIP file
3. Open Chrome → `chrome://extensions/`
4. Enable "Developer mode" (top-right)
5. Click "Load unpacked"
6. Select the extracted folder

### For Edge:
1. Download the [Extension ZIP](https://github.com/3issam-hub/Pdf-Extension.git)
2. Extract the ZIP file
3. Open Edge → `edge://extensions/`
4. Enable "Developer mode" (bottom-left)
5. Click "Load unpacked"
6. Select the extracted folder
